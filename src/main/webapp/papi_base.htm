General information about this API usage

Our API allow you to programmatically access datasets collected and analyzed by our technologies. Our goal is to facilitate a wide range of uses: from custom link lists to complex neural networks.

Basics

To start working with the API you don't need to register for api-key or anything else: we do not have it, - all you need is just start using it! That much it is simple!

All API methods are located at https://qvim.kostya.sexy/api/METHOD (some of them require additional arguments). All methods can be accessed both within HTTP/HTTPS protocols, and also both accepting GET/POST requests. The result to your request is a json string. Whether there is an error corresponding to your request, you will notify parameter error in resulting json, otherwise you will get the one that is described on method page.
